Como usar a Calculadora CLT vs PJ

Entradas

  • CLT: Salário bruto.
  • PJ: Faturamento, RBT12, Anexo, Regime, Pró-labore (opcional).

Resultados

  • CLT: Líquido mensal + detalhamento (INSS, IRRF, FGTS depósito).
  • PJ: Líquido = faturamento − DAS − INSS PL − IRPF PL − CPP.

Dicas

  • RBT12 real para alíquota efetiva correta.
  • Sem pró-labore, usamos 28% do faturamento.

Notas importantes

  • Fator R → Anexo: se Fator R ≥ 28%, use Anexo III; se menor, Anexo V. Calcule o Fator R.
  • Pró-labore e CPP: pró-labore sofre INSS 11% (até o teto) e IRPF progressivo. CPP 20% aplica em Simples IV / Presumido / Real (ou se você escolher Anexo IV).
  • Escopo: não inclui VR/VA/VT, plano de saúde, bônus e custos do empregador. DAS detalhado apenas para Anexos III e V.
